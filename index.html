Academic Integrity


Our Operating System has the following features.

Required:

  • GDT, LDT, IDT set up
  • PIC i8259 Interrupt Handler
  • Exception Handler
  • Keyboard
    • alphabetics & numbers & partial punctuations
    • function keys: ctrl, alt, shift, capslock
    • ctrl + L: clear the screen
  • RTC (real-time clock)
  • Virtual-Physical memory paging
  • Terminal Driver (doesn't exist in hardware)
  • Read-only Filesystem
  • Test cases that TAs enjoy, deprecated in Checkpoint 3
  • Process Control Block
  • Basic Systemcall Support (a lot of work)
    • execute()
    • halt()
    • open()
    • close()
    • read()
    • write()
    • Systemcall Assembly Linkage
  • Program Loader
  • User-Level Code Support
  • User-Level Argument Support
  • Video Memory Map
  • Creating/Exiting Shells
  • Switching among three terminals
  • Round-Robin Scheduler using PIT

Extra:

  • some
  • Mouse Support
    • Button-click detection
    • DPI control
    • Blinking
    • Change style when clicking left button
    • Self-designed pattern
  • Keyboard Related
    • ctrl + C: halt current process
    • ctrl + V: emerge all invisible terminals
    • ctrl + N: Send UDP packet to Google server with content “Hello”
    • ctrl + P: piano mode on/off
    • ctrl + Q: recursive halt
    • Tab auto compliment
    • upp/down arrow: command histroy
  • Graphic User Interface (GUI)
    • Screen background generated using a random algorithm
    • Displaying three terminals at the same time
    • Moving terminals around using mouse
    • Terminal visibility
    • Terminal button control (minimize and recursive halt)
    • Gaussian blur on top bar
    • Change resolution - Alt + F1/F2/F3
    • Change font type - Alt + F4/F5/F6
    • Very High resolution support
    • Time on the upper right corner
    • "Lucky Font"
    • Window size is auto-adjusted according to font size
  • Networking
    • PCI controller
    • PCI Brute-force Scanner
    • RTL8139 driver
    • host-byte order & network-byte order converting
      • htonl()
      • htons()
      • ntohl()
      • ntohs()
    • Link Layer
      • Ethernet Protocol
      • Address Resolution Protocol
    • Network Layer: IPv4
      • Header Format
      • Checksum calculation
    • Transport Layer: UDP
  • Sound
    • Screen background generated using a random algorithm
    • Displaying three terminals at the same time
    • Moving terminals around using mouse
    • Beep using PC speaker
    • Keyboard Piano supporting from C4 to D#6/Eb6
